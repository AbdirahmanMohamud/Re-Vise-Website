Memory

RAM vs ROM

RAM(Random Access Memory) ROM(Read Only Memory)
Volatile so is completely wiped when the computer is turned off The data is stored permanently on the ROM so is never removed
The RAM is only useable after the operating system has been loaded Used in the start-up process of a computer as the operating system is stored in the ROM

The purpose of ROM in a computer system is to store permanent instructions which is used for performing input and output tasks. Also, ROM is a storage medium for data that can only be read and not manipulated in any form. It is for this reason why the operating system is in the ROM.

The purpose of RAM in a computer system is to store instructions from the operating system to application programs and general data currently in use. However the data in the RAM stays there only as long as your computer is running.

Virtual Memory

Virtual memory is when the computer has too much data to process and the RAM is full. Since the computer doesn’t have enough memory to process the data, some data is processed from ‘virtual’ memory that is inside of the hard disk drive. The way that this works is that the least recently used stored data is swapped from the RAM to the virtual memory and when is needed, is swapped back into the RAM.

Virtual Memory

Flash memory or solid state memory stores data electrically and works by being developed from a type of ROM called ‘electrically erasable programmable read-only memory’ or (EEPROM). The way that flash memory works is that a flash of electric current pushes electrons through a barrier and traps them in position to be read by the computer.

Dynamic vs Static RAM

There are two types of RAM:

• Dynamic RAM is 1 transistor and 1 capacitor for each bit – hold a small charge for a few milliseconds then needs to be refreshed. Power is always needed.

• Static RAM is 4-5 transistors wired together for each bit of data but it doesn’t lose its charge so there is no need to refresh every few milliseconds. Power is always needed. It is faster than Dynamic RAM but more expensive
