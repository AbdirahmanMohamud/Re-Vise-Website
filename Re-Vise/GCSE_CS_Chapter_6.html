System Security

There are numerous ways for a hacker to get into a system. These ways include:

• Malware is any software which can harm a computer or user. These can include viruses, spyware, adware and pharming:

o Virus entails three versions. These include standard virus which hide in files or programs and replicate themselves to delete or damage data. There are Worms Virus which don’t damage data but replicate themselves until the computer slows down and makes it useless. Finally, there is a Trojan virus which works by having a program such as a game which works fine but in the background is deleting files and creating backdoors to allow other users to gain access to your system

o Spyware aims to not delete files but instead send back information such as passwords, usernames, websites the user visits and recent purchases.

o Adware downloads and displays pointless adverts and collects marketing information.

o Pharming seeks to change the IP address stored in the DNS server to another IP address so that when someone types the URL to go to Google, instead it will go to Bing as an example.

• Phishing seeks to acquire sensitive information about a user such as usernames, passwords and bank details and this is done by direct electronic communications such as an email or a phone call. An example could be that a user gets an email from what they believe to be PayPal but instead, it is a fake email and they have your credit card details, username and password.

• Social Engineering is the act of manipulating people into inadvertently giving their details to a hacker which can jeopardise the entire network for a company

• Brute force attacks is when criminals simply use trial and error to hack an account by using a massive amount of combinations in order to login

• Denial of service attacks (DDoS) works by acquiring a large amount of the network simultaneously so that the servers can’t comprehend the sheer amount of sudden traffic that the network goes down and the criminals demand money to stop the bombardment.

• Data interception and theft is when hackers use specialist hardware and software to monitor network traffic and intercept packets containing sensitive data. These packets are then assembled to own passwords.

• SQL injection is when SQL is added to the end of a password and alternating this SQL then results in being able to access the accounts of other users

• Poor network policy is when a lack of important rules or procedures aren’t available so the risk of a breach increases exponentially

And ways that these attacks can be identified or prevented include:

• Penetration testing is when a company will invite experts to simulate an attack to find any holes in their firewalls or policies

• Network forensics is software that consistently monitors network traffic in order for a response to a sudden attack to be as swift as possible

• Network policies is when a company will have stringent rules that need to be followed in order to access computers such as complex passwords, different levels of access, no use of USB sticks, no installation of software or files and having computers immediately lock when the user has left the area.

• Anti-malware software can identify and destroy and form of virus imminently so that it can’t delete important information without being destroyed

• Firewalls works by the knowledge that every computer is made up of 256 ports and a firewall monitors the data that goes through each port. Ports will be open when data needs to be sent through and closed when it isn’t necessary for them to be open.

• Encryption is where data is scrambled before being sent across a network so that it is unreadable if intercepted without an encryption key which converts ‘plain text’ into ‘cipher text’ and back to ‘plain text’.
