Computational Logic

Computers use binary to store a value of either 0 or 1 which is a computer means either a transistor is on (1) or off (0). Computers are made up of billions of transistors which can individually form to produce a logic gate. There are three main types of logic gates:

o AND gate means that both inputs have to be on in order for a value to pass through as the name implies and without it, then nothing is passed through as a signal

o OR gate means that only one input has to be on in order for the circuit to function as the name would suggest.

o NOT gate means that the opposite of what is entered is outputted as the name suggests
