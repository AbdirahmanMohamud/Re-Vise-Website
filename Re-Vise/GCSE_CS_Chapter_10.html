Programming Techniques

Sequence:

Is in a program and is how it leads to the next ordered action in a predetermined order and this is typically done with an array

Selection:

Also called a decision and is when a program differs depending on an input and is made by an if/else statement

Iteration:

This is when in a program, certain elements are repeated so that it can perform a certain task. In python, this is commonly done with a ‘for’ loop or a ‘while’ loop

String Manipulation:

This is when an input is manipulated in one of many ways to fit a certain purpose of the programmer. There are numerous examples of this which include:

• Len(variable) to show how long a string is in terms of how many characters there are

• (variable[number of position]) to find the letter at that specific location in a string

• Variable.isupper() to find if the variable is in all caps

• Variable.islower() to find if the variable is in all lowercase

• Variable.isalnum() to find if the variable is alphanumeric

• Variable.isalpha() to find if the variable contains only letters

• Variable.isdigit() to find if the variable contains only numbers

File Handling:

In Python, you are able to store data in text files outside of the program and to write into those files mid program and to read from those files in a program to output them in the program itself. The way that this is done is like so, if you want to write into a text file:

1. First you create the file and open it in write mode using the ‘open()’ function like so:

Newfile = open(“The New File.txt”,”w”)

2. Then you can write into the file as like so:

Newfile.write(“Hello World”)

3. Finally, you have to close the file like this:

Newfile.close()

Now if you want to read from the file, this is what you do:

1. You need to open the file in Python using the ‘open()’ function:

Newfile = open(“The New File.txt”)

2. You then can read the file with the following:

Print(newfile.read())

3. The output should be what has already been entered into the file such as:

Hello World

SQL:

SQL is a language that is commonly used in databases in the manipulation of databases to add, remove and change. The way that this is typically done in Python to connect a database is like so:

1. First you need to import the function of SQL like so:

Import sqlite3

2. Then you need to connect the python file to the SQL database like so:

New_db = sqlite3.connect(“new_db.db”)

3. Next, you need to use a cursor object to start working with the SQL:

C = new_db.cursor()

4. Now you can execute an instruction in Python like so:

c.execute(‘’’CREATE TABLE Students

(student_firstname text,

Student_surname text,

Student_DOB text,

Form text)

‘’’)

5. Finally, you need to commit and close the SQL program by committing the changes to the program like so:

New_db.commit()

New_db.close()

Arrays:

An array is an example of a data structure which allows us to store more than one item of data together under one identifier which can be used to access the individual items of the data. An array is a collection of data which has the same data type. A list can contain data that isn’t all the same data type. An array uses a static area of memory whilst a list uses memory dynamically so different areas of memory can be used. The way that a list can be formed:

List = [“Train”, ”Car”, “Foot”, “Bus”, “Taxi”]

You can also change lists in many ways, such as:

• List.reverse() to reverse the order of the list

• List.sort() to sort the list into alphabetical or numerical order

• List.append() to add items on to the end

• List.remove() to remove items from the list

• List.index() to find the position of an item on the list

• List.pop() to ‘Pop’ an item out of the list

Data Types

Data is classified into types, such as a set of whole numbers (also known as integers) or a set of printing characters.

Different types of data are represented in different ways inside a computer and need varying amounts of memory to store them. They also have different operations that can be performed upon them. All values that belong to the same data type will be represented in the same way.
