Algorithms

Computational Thinking (CT condensed) is an approach to problem solving and is about taking a complex problem and making it easier. There are 3 main techniques that underpin computational thinking:

• Abstraction which is about moving the real to virtual. An example would be recreating a cat digitally. You would focus on the main details instead of the tiny parts.:

• Decomposition is simply breaking down a massive problem into a set of sub-problems that are more manageable to deal with instead of the massive problem:

• Algorithmic Thinking relates to algorithms which are procedures used to get the intended result. Examples of algorithmic thinking would include pseudocode and flowcharts:

In addition to computational thinking, there are algorithms that used for searching and sorting, these include::

• Linear Search which works by searching through a set of data, one item at a time until the desired data is found:

• Binary Search (a.k.a Divide and Conquer) is when you search at the middle to see if the data is lower than the middle or higher and then divide the data by two to remove half of the undesired data. This division is continuous until the item is found.:

Also, there are three types of sorting algorithms which are:

• Bubble sort which works by going through one piece of data at a time and seeing if it is higher than the other, if it is, then the data moves to be in ascending order. This is done over several cycles of going through the data and it is the slowest form of sorting algorithm.:

• Insertion Sort is more efficient than the Bubble Sort algorithm as it sorts with every previous number it has seen so that it only needs one cycle to sort the entire set of data.:

• Merge Sort works by having the data match into pairs and then sorting them in a similar manner to a bubble sort algorithm. An example would be that 1, 2,3,4,5,6,7,8 is paired to that the numbers are 12,34,56,78. After this, the numbers are sorted into one chain in order.:
