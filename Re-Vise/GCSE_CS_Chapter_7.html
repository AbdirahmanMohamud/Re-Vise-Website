System Software

There are two types of system software:

• Operating System Software

• Utility Software

The Operating System works in between applications and hardware and allows the two to work by corresponding functions in the applications to certain areas of the hardware of a computer.

The Kernel is the heart of the operating system and is responsible for looking after ‘the most low-level hardware operations’. It is what applications make use of to operate the computer’s hardware.

Without an operating system, applications would not be able to communicate with the computer’s hardware. The operating system bridges the gap. It gets the applications and hardware to work together. An example would be that without an operating system, a music player would not be able to talk to the speakers and no sound would be produced. The operating system would pass instructions from the application to the speakers to allow sound to play.

There are 5 major roles of the operating system:

• Memory Management (The OS manages the RAM so that when you load an application, the OS will copy the application to the RAM. The OS allows ensures that each program has its own space on the RAM)

• Peripheral Management (Peripherals are devices that are attached to a computer such as a printer or speakers and the OS controls these devices and gives programs access to them)

• Multi-Tasking is relating to how the CPU is being controlled by the OS to manage multiple programs at once by as before the only way that the CPU would work was either allowing processes to take it in turn or getting the shortest job done first.

• Security works as OS’ manage their user by asking for usernames and passwords. Also the primary user of the computer has certain permissions that other users don’t.

• User Interface (the word interface means ‘coming together’ and in computers is allowing a user to interact with a computer. Operating Systems provide users with an interface so that they can interact with the computer.) These interfaces can be divided into three types:

o Graphical User Interface or GUI is the most modern form of user interface and is how most OS’ interact with the user. It uses windows, icons, menus and pointers to control the computer and has been around since the mid-1980s.

o Menu Driven Interfaces are newer than the other two but more derivative than the GUI as instead of WIMP being used, only menus can be used to access information and this type of interface is typically used in early mobile phones and MP3 players or games consoles.

o Command Line Interface is the most derivative of the three and the oldest of the three due to only seeing information in an interface without WIMP or menus so it’s the least aesthically pleasing interface of the three.

Utility Software works with Encryption Software (which facilities the act of encrypting software so that any intercepted data can’t be used against the user), Formatting Software (refers to the storage device for data storage and the creation of sectors and tracks on which data can be stored), Defragmentation Software (to stop long load times by positioning data that are related to one another), Data Compression Software (which is when data is compressed one of two ways in order to save on file size) and Backup Software(which is when copies of data is made stored elsewhere of the computer, typically in the cloud, either completely or incrementally so that if the computer is damaged, the data is still there).
